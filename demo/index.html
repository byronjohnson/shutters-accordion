SHUTTERS

a CSS accordion utility

lightweight, developer-friendly accordion component that prioritizes simplicity and customization.

Basic Accordion Demo

What is the Shutters CSS Accordion?

The Shutters CSS Accordion is a lightweight utility that provides accordion functionality using modern CSS and JavaScript. It's designed for easy integration into any web project.

How does it work?

The accordion uses modern JavaScript combined with CSS Grid for smooth animations. Any element with proper ARIA attributes can control the expand/collapse states.

  • Modern JavaScript with flexible HTML structure
  • CSS Grid animations for smooth transitions
  • Full accessibility support with ARIA
What are the key features?

Key features include:

  • Lightweight: Clean and minimal code
  • Modular: Separate core functionality from presentation styles
  • Accessible: Full ARIA support and keyboard navigation
  • Customizable: Black and white theme with CSS custom properties
  • Compatible: Works with Vite and modern build tools

How to Use

1. Installation

Include the necessary CSS and JavaScript files in your project:

<link rel="stylesheet" href="shutters-core.css" />
<script type="module" src="shutters-core.js"></script>

Optional: Include the theme stylesheet if you want to match the demo styling:

<link rel="stylesheet" href="shutters-theme.css" />

2. HTML Structure

Create your accordion with the following HTML structure:

<div class="shutters-accordion">
  <div class="shutters-item">
    <div class="shutters-header" role="button" tabindex="0" 
         aria-expanded="false" aria-controls="panel-1">
      <span class="shutters-title">Section Title</span>
      <span class="shutters-icon"></span>
    </div>
    <div class="shutters-content" id="panel-1">
      <div class="shutters-body">
        <p>Your content goes here</p>
      </div>
    </div>
  </div>
</div>

3. Initialize JavaScript

Initialize the accordion with optional configuration:

import { ShuttersAccordion } from './shutters-core.js';

const accordion = new ShuttersAccordion({
  container: '.shutters-accordion',
  animationDuration: 300,
  animationEasing: 'ease-in-out',
  defaultOpen: 'none'
});

4. Configuration Options

container
CSS selector, element, or array of containers (default: '.shutters-accordion')
animationDuration
Animation speed in milliseconds (default: 300)
animationEasing
CSS easing function (default: 'ease-in-out')
defaultOpen
Default open items: 'first', 'all', 'none', or array like [0, 2]

5. Programmatic Control

Control accordion items programmatically using the API:

// Open specific item by index
accordion.open(0);

// Close specific item by index
accordion.close(2);

// Toggle specific item by index
accordion.toggle(1);

// Destroy the accordion instance
accordion.destroy();

6. Auto-Close Mode

Add the shutters-autoclose class to enable auto-close behavior:

<div class="shutters-accordion shutters-autoclose">
  <!-- Only one item can be open at a time -->
</div>

7. Customization

Override CSS custom properties to customize the appearance:

.shutters-accordion {
  --shutters-bg-color: #fff;
  --shutters-border-color: #000;
  --shutters-focus-color: #0066cc;
  --shutters-hover-bg: #f5f5f5;
  --shutters-text-color: #000;
  --shutters-animation-duration: 0.5s;
  --shutters-animation-easing: cubic-bezier(0.4, 0, 0.2, 1);
}
